Sobre mim

Sou um desenvolvedor Front-end Júnior, buscando uma vaga de emprego. Me dedico diariamente em programação, participo de comunidades voltadas a esse assunto como a DevEmDobro e a Rocketseat. Faço aplicações com base em sites da internet, como por exemplo o FrontEnd Mentor ou das comunidades que estou inserido. Dentro desses projetos, faço o uso da linguagem de programação JavaScript para parte lógica, e utilizo a linguagem de marcação HTML e de estilização CSS para a conclusão deles. Tenho aplicações feito com a biblioteca React, exemplo foi de uma API que consumi para pegar dados de usuário do GitHub. Me sinto prepara para o mercado, sou uma pessoa comunicativa, gosto de ajudar quem está ao meu redor e quero muito conseguir minha vaga como Front-end Júnior.
